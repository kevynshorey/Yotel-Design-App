A) Tab Navigation — Module Switcher

Top tabs switch between modules. Each module is a full-page experience. Clean separation. The 3D massing tool gets full viewport width. Investor portal is a separate tab with its own layout.

Pros

  • Each module gets maximum screen space
  • Simple mental model — one thing at a time
  • Investor tab can have totally different layout

Cons

  • Can't see financials while designing
  • Context switching between modules

B) Command Center — Side Rail + Panels

Icon rail on the left switches context. Main viewport dominates. Right side shows live summary cards from ALL modules simultaneously. Click a card to expand into that module's detail view. Always-visible project health.

Pros

  • See financial impact of design choices in real-time
  • Professional "mission control" feel for investor demos
  • Never lose sight of project health

Cons

  • More complex to build
  • Summary cards need careful data flow

C) Dashboard + Workspaces

Three modes: DESIGN (massing tool full-screen), MANAGE (planning + dataroom grid), INVEST (financial + investor portal). Dashboard home shows summary tiles from all modules. Each workspace is purpose-built for its audience.

Pros

  • Different layouts optimized per audience
  • Dashboard gives executive overview
  • INVEST workspace = instant investor demo

Cons

  • Three different layouts to build and maintain
  • Risk of feature fragmentation
