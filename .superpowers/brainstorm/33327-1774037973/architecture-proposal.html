Module Breakdown

Module 1: 3D Massing Tool (Priority 1)

TypeScript rewrite of Python engine

  • Parametric form generator (BAR, L, U, C forms)
  • Room layout calculator with brand rules
  • Multi-criteria scoring (9 weights)
  • Planning + brand validation engine
  • Three.js 3D viewer with basemap tiles
  • Runs CLIENT-SIDE for instant feedback
  • Option comparison + export

v2 financials baked in: $350/sf, 51% GOP, $40M TDC

Module 2: Planning Compliance (Priority 2)

Interactive regulatory tracker

  • 5 approval tracks (A-E) with step status
  • Document checklist per track
  • Design controls dashboard
  • Setback visualizer (road class → setback)
  • Compliance score per design option
  • Timeline / critical path view

Pulls from PLANNING_COMPLIANCE.md data

Module 3: Financial Dashboard (Priority 3)

Live proforma & sensitivity

  • 10-year proforma with v2 parameters
  • Capital stack waterfall visualization
  • Sensitivity sliders (ADR, occ, cap rate, GOP)
  • DSCR / IRR / LP multiple real-time calc
  • TDC build-up breakdown chart
  • Revenue ramp visualization

Connected to massing tool — design changes update financials

Module 4: Project Dataroom (Priority 4)

Interactive document management

  • Document register with tier classification
  • Risk register with severity tracking
  • Open items board (Kanban-style)
  • Upload / categorize new documents
  • Audit trail for resolutions

Pulls from MD files we created

Module 5: Investor Portal (Priority 5)

Shareable project summary for LPs, DFIs, lenders

    • One-page project overview with key metrics
    • Interactive 3D model (read-only, best option)
    • Financial summary with return projections
    • Planning progress timeline
    • Time-limited signed share links
    • View tracking (who looked, when)

The "pitch in a URL" — send a link, investor sees the full story
